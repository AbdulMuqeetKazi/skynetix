# Skynetix

Skynetix is an emergency orchestration system designed to help users reach the right help reliably, even when connectivity or systems fail.

## Core Principles
- Offline safety first
- AI assists, never blocks
- Telecom > Internet > AI
- One responder per incident
- Escalation over silence

## Project Structure
- mobile-app/ → Flutter client
- backend/ → Node.js orchestration services
- shared/ → Common models and enums
- docs/ → Locked specifications
